<Callout type="info" title="Para equipas de clubes">
Este centro de ajuda foi pensado para quem usa o ClubIQ BackOffice no trabalho diário. O foco está no
que pode fazer no produto, onde encontrar cada área e o que esperar de cada fluxo.
</Callout>

Use estes guias quando precisar de concluir uma tarefa operacional, perceber como um fluxo funciona ou
ajudar alguém novo a aprender a plataforma.

## O que vai encontrar aqui

- Guias passo a passo para as principais áreas do BackOffice
- Explicações de ecrãs, separadores, estados e comportamentos automáticos
- Notas operacionais para ajudar a equipa do clube a trabalhar com confiança
- Ligações entre temas relacionados para tornar os fluxos mais fáceis de seguir
