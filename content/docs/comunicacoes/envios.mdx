O separador **Envios** é a porta de entrada para uma comunicação manual. Não mostra uma lista: apresenta um cartão de
arranque que abre o assistente de composição.

## O que precisa antes de começar

- permissão para **enviar** comunicações;
- pelo menos um destinatário elegível;
- um conteúdo pronto, criado no momento ou a partir de um modelo existente.

<Callout title="Permissões diferentes">
No módulo atual, escrever conteúdo, manter modelos e disparar uma comunicação não são exatamente a mesma coisa. Pode
ter acesso à área e mesmo assim não ver o botão para enviar se lhe faltar a permissão específica de envio.
</Callout>

## Como enviar uma comunicação manual

<Steps>
<Step>
Abra **Comunicações → Envios**.
</Step>
<Step>
Clique em **Enviar** para abrir o assistente de composição.
</Step>
<Step>
Escolha o canal e o conteúdo a usar. Se arrancar a partir de um modelo, o assistente pode vir já pré-preenchido.
</Step>
<Step>
Defina a audiência a partir de um segmento guardado ou de filtros manuais.
</Step>
<Step>
Revise o resumo final e confirme o disparo.
</Step>
<Step>
Abra [Histórico](/docs/comunicacoes/historico) para validar o resultado real do envio.
</Step>
</Steps>

## Boas práticas

- use um segmento mínimo quando estiver a testar;
- valide primeiro o modelo antes de enviar para uma audiência ampla;
- confirme sempre o envio no histórico, não apenas no assistente.
