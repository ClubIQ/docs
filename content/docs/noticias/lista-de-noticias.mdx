A **lista de notícias** é a vista principal do módulo. Mostra capa, título, estado, audiência, data e contadores de
leitura quando aplicável.

## Estados usados na lista

- **Rascunho**
- **Agendada**
- **Publicada**
- **Arquivada**

## Ações comuns

- abrir para editar;
- pré-visualizar;
- duplicar;
- publicar;
- retirar publicação;
- arquivar;
- eliminar.

## Gestão em lote

A lista também permite seleção múltipla, útil quando precisa de arquivar ou eliminar vários itens de uma vez.
